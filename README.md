# Sistema de Gestão - Estúdio de Tatuagem

Um sistema simples e direto de linha de comando (CLI) desenvolvido para demonstrar conhecimentos práticos em Banco de Dados Relacional e integração com Back-end. 

## Tecnologias Utilizadas
* **Python 3:** Construção da lógica de aplicação e menus interativos.
* **SQLite3:** Banco de dados relacional nativo e super leve para armazenamento local.
* **SQL:** Utilização de comandos estruturados para manipulação de dados (CRUD completo).

## Funcionalidades e Aprendizados
* **Modelagem de Dados:** Criação de tabelas com `PRIMARY KEY` e relacionamentos usando `FOREIGN KEY`.
* **Consultas Avançadas:** Uso de `INNER JOIN` para cruzar dados de agendamentos e categorias de arte.
* **Segurança:** Prevenção ativa contra SQL Injection utilizando consultas parametrizadas (`?`).
* **Persistência:** Geração automática do arquivo físico `.db` na inicialização do sistema.
